# 生长评估功能 - 数据来源说明文档

## 📋 概述

为了增强用户对生长评估功能的信任度和理解，我们在界面上增加了详细的数据来源说明，让家长了解评估标准的权威性和可靠性。

## 🎯 核心改进

### 1. 权威数据来源卡片

在生长曲线图表下方新增了详细的数据来源说明卡片，包含：

#### 标准信息
- **标准名称**: WS/T 423-2022《7岁以下儿童生长标准》
- **发布机构**: 中华人民共和国国家卫生健康委员会
- **实施日期**: 2023年3月1日
- **评价方法**: 标准差（SD）法
- **数据基础**: 基于中国儿童生长发育调查数据

#### 适用说明
- 适用于 0-7 岁中国儿童
- 替代旧标准 GB/T 31179-2014

### 2. 增强的帮助对话框

点击"如何解读生长曲线？"按钮，用户可以看到：

#### 标准差与百分位对应关系
```
• -3SD ≈ P0.1：极低
• -2SD ≈ P3：低于正常范围下限
• -1SD ≈ P16：中下水平
• 0SD = P50：中位数（正常）
• +1SD ≈ P84：中上水平
• +2SD ≈ P97：高于正常范围上限
• +3SD ≈ P99.9：极高
```

#### 评价标准
- ✅ -2SD ~ +2SD：正常范围
- ⚠️ <-2SD 或 >+2SD：需要关注
- 🚨 <-3SD 或 >+3SD：建议就医

#### 营养状况评价（依据 WS/T 423-2022 表3）
- 体重-年龄 <-2SD：低体重
- 身高-年龄 <-2SD：生长迟缓
- 体重-身高 >+2SD：超重
- 体重-身高 >+3SD：肥胖

#### 重要提示
强调本功能仅供参考，不能替代医学诊断，如有异常需及时咨询专业医生。

### 3. 无数据状态优化

当用户还没有添加任何生长数据时，显示：
- 友好的空状态插图和提示
- 清晰的功能说明
- **权威标准徽章**："采用国家卫健委 WS/T 423-2022 标准"

### 4. UI 设计细节

#### 视觉层次
- 使用图标增强信息的可识别性
- 卡片式布局，信息分组清晰
- 颜色语义化：
  - 绿色 = 权威/安全/正常
  - 蓝色 = 信息/中立
  - 橙色 = 警告/需关注
  - 红色 = 危险/需就医

#### 信息图标
- 🛡️ 盾牌图标：代表权威和可信
- 📄 文档图标：标准名称
- 🏢 建筑图标：发布机构
- 📅 日历图标：实施日期
- ⚗️ 烧瓶图标：评价方法

## 📱 用户体验提升

### 建立信任
1. **透明度**：完整展示数据来源的所有关键信息
2. **权威性**：突出国家卫健委官方标准
3. **时效性**：明确标注最新标准的实施日期

### 教育价值
1. **科普知识**：帮助家长理解标准差法和百分位的含义
2. **评价标准**：清晰说明什么是正常、什么需要关注
3. **实用指导**：提供具体的营养状况评价标准

### 减少焦虑
1. **正确解读**：避免家长因不理解数据而过度焦虑
2. **明确界限**：清楚告知何时需要咨询医生
3. **免责声明**：强调仅供参考，不替代医学诊断

```typescript

  ├─ 生长曲线图表
  ├─ 图例说明
  ├─ 数据来源卡片 ⭐ 新增
  │   ├─ 权威标志
  │   ├─ 标准信息
  │   ├─ 发布机构
  │   ├─ 实施日期
  │   ├─ 评价方法
  │   └─ 适用说明
  └─ 帮助按钮
```

- 使用浅色背景 `#F9FAFB` 区分信息区域
- 边框 `#E5E7EB` 增加层次感
- 图标背景 `#EBF5FF` 统一视觉风格
- 响应式字体大小，适配不同屏幕
