# BabyBeats 资源文件

## 应用图标

### icon.png
- **尺寸**: 1024 x 1024 px
- **格式**: PNG
- **用途**: 
  - iOS App 图标
  - Android App 图标
  - App Store 展示图标
  - 启动画面

**设计说明**:
- 可爱的宝宝头像设计
- 粉色背景，温馨柔和
- 包含成长曲线元素，象征记录和成长
- 爱心元素，传递关爱之情
- 符合 App Store 和 Google Play 的图标规范

## 图标要求

### App Store (iOS)
- 尺寸: 1024 x 1024 px
- 格式: PNG（无透明通道）
- 无圆角（系统会自动添加）
- 色彩空间: sRGB 或 Display P3

### Google Play (Android)
- 尺寸: 512 x 512 px（会自动从1024x1024缩放）
- 格式: PNG
- 支持透明通道
- 自适应图标支持

## 启动画面

当前配置使用同一个图标作为启动画面：
- 背景色: #FFB6C1 (浅粉色)
- 图标居中显示
- contain 模式（保持比例）

## 未来扩展

如需更换图标，请确保：
1. 尺寸为 1024 x 1024 px
2. PNG 格式
3. 高质量，清晰可辨
4. 符合应用主题
5. 在小尺寸下依然清晰（测试在 60x60, 120x120 等尺寸）
