# 白噪音音频文件

## 所需文件

请将以下白噪音音频文件放置在此目录：

1. `white-noise.mp3` - 白噪音
2. `rain.mp3` - 雨声
3. `ocean.mp3` - 海浪声
4. `heartbeat.mp3` - 心跳声
5. `fan.mp3` - 风扇声
6. `shush.mp3` - 嘘声

## 音频文件要求

- **格式**: MP3 或 M4A
- **时长**: 建议 30秒 - 5分钟（将自动循环播放）
- **采样率**: 44.1kHz
- **比特率**: 128kbps - 192kbps
- **声道**: 立体声（Stereo）

- 需要注册账号

### 3. 使用音频生成工具
- Audacity（免费开源）: 可以生成白噪音、粉红噪音
- 在线生成器: https://mynoise.net

## 制作自己的白噪音

使用 Audacity 生成白噪音：

1. 下载安装 Audacity
2. 菜单：生成 > 噪声
3. 选择"白噪音"类型
4. 设置时长（建议2-3分钟）
5. 调整音量（建议 -20dB 到 -10dB）
6. 导出为 MP3 格式

## 临时方案

在没有音频文件的情况下，应用会使用以下方式：
- iOS: 使用系统提示音作为占位
- 自定义录音功能仍然正常工作
- 建议尽快添加实际的白噪音文件以获得最佳体验

## 添加音频文件后

1. 将音频文件复制到 `assets/sounds/` 目录
2. 重新构建应用
3. 音频会被打包到应用中，无需网络即可使用
