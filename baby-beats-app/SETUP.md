## 快速启动

### 1. 安装依赖
```bash
cd baby-beats-app
npm install
```

### 2. 启动项目
```bash
# iOS 模拟器
npm run ios

# Android 模拟器
npm run android

# Web 浏览器
npm run web

# 或者启动开发服务器后扫码
npm start
```

### 3. 测试功能

应用启动后会自动：
1. 初始化 SQLite 数据库
2. 创建一个测试宝宝档案
3. 进入首页（Today Screen）

你可以：
- 查看首页的今日统计
- 点击右上角 "+" 按钮添加喂养记录
- 在喂养页面测试计时器功能
- 记录不同类型的喂养
