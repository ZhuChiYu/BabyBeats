# 🎊 新功能使用指南

欢迎使用 BabyBeats 健康管理和里程碑记录功能！

## 🏥 健康管理功能

### 1. 体温记录 🌡️

**使用场景**：宝宝发烧时记录体温变化

**操作步骤**：
1. 点击首页 "+" 按钮 → 选择 "体温"
2. 输入体温值（自动显示状态：正常/低烧/发烧）
3. 选择测量方式（额温/耳温/腋温等）
4. 选择测量时间
5. 可选：添加症状备注
6. 点击 "保存"

**查看记录**：
- 健康Tab → 体温记录卡片
- 或点击体温卡片进入列表页

**颜色标识**：
- 🟢 正常：< 37.3°C
- 🟠 低烧：37.3-38.0°C
- 🔴 发烧：38.0-39.0°C
- 🔥 高烧：> 39.0°C

---

### 2. 疫苗接种 💉

**使用场景**：记录疫苗接种，管理接种计划

**操作步骤**：
1. 点击首页 "+" 按钮 → 选择 "疫苗"（或进入健康Tab）
2. 从常见疫苗列表选择（或自定义输入）
   - 卡介苗、乙肝疫苗、百白破等15种常见疫苗
3. 填写接种日期
4. 可选信息：
   - 剂次（第几针）
   - 接种地点
   - 批次号
   - 下次接种日期（自动提醒）
5. 点击 "保存"

**查看记录**：
- 健康Tab → 疫苗接种卡片
- 显示：已接种数量 | 待接种数量

**提醒功能**：
- 设置下次接种日期
- 开启提醒（接种前7天通知）

---

### 3. 看病记录 🏥

**使用场景**：记录每次看病的详细信息

**操作步骤**：
1. 点击首页 "+" 按钮 → 选择 "就诊"
2. 填写基本信息：
   - 就诊时间
   - 医院名称
   - 科室（可快速选择常见科室）
   - 医生姓名（可选）
3. 填写病情：
   - 主要症状（可点击标签快速输入）
   - 诊断结果
   - 医嘱建议
4. 可选：添加备注
5. 点击 "保存"

**查看记录**：
- 健康Tab → 就医记录卡片
- 显示：总就诊次数 | 近30天就诊

**快速输入**：
- 常见症状标签：发烧、咳嗽、腹泻、皮疹等18种
- 常见科室：儿科、儿科门诊、小儿呼吸科等15个

---

### 4. 用药记录 💊

**使用场景**：记录用药情况，管理疗程

**操作步骤**：
1. 点击首页 "+" 按钮 → 选择 "用药"
2. 选择药品：
   - 从常用药品列表选择（自动填充剂量）
   - 或自定义输入
3. 填写用药信息：
   - 用药剂量（如：5ml、1片）
   - 用药频次（如：每日3次）
   - 给药方式（口服/外用/注射/其他）
4. 设置疗程：
   - 开始日期
   - 结束日期（可选）
5. 可选：关联就诊记录
6. 点击 "保存"

**查看记录**：
- 健康Tab → 用药记录卡片
- 显示：总记录数 | 进行中

**常用药品**：
- 退烧药：布洛芬、对乙酰氨基酚
- 感冒药、止咳药
- 肠胃药：妈咪爱、蒙脱石散
- 外用药、维生素等

## ⭐ 里程碑记录功能

### 使用场景
记录宝宝成长过程中的珍贵瞬间：第一次微笑、第一次叫妈妈、第一次走路...

### 操作步骤

**记录里程碑**：
1. 点击首页 "+" 按钮 → 选择 "里程碑"
2. 选择类别（8大类别）：
   - 🏃 运动发展：抬头、翻身、爬行、走路...
   - 💬 语言发展：微笑、说话、叫爸妈...
   - 👶 社交发展：认人、拍手、分享...
   - 🍽️ 饮食发展：吃辅食、用勺子...
   - 🛏️ 生活技能：穿衣、刷牙...
   - ⭐ 特殊时刻：第一次洗澡、理发、旅行...
   - 💡 认知发展：认颜色、数数、认字...
   - ❤️ 情感发展：笑出声、说爱你...

3. 选择具体事项（60+预设选项）
   - 或自定义输入
4. 选择发生时间
5. 添加照片（可选）
   - 从相册选择
   - 支持裁剪和压缩
6. 填写详细描述（可选）
7. 点击 "保存"

**查看里程碑**：
1. 进入 "成长" Tab
2. 点击 "成长里程碑" 卡片
3. 浏览精美的时间轴展示
   - 垂直时间轴布局
   - 彩色分类标记
   - 照片展示
   - 按时间倒序排列

## 💡 使用技巧

### 快速记录
1. **使用快速操作菜单**：
   - 首页右上角 "+" 按钮
   - 一键打开所有记录功能
   - 8个快捷入口（2x4网格）

2. **利用预设列表**：
   - 疫苗、药品、症状都有常用列表
   - 点击即可快速填入
   - 减少输入时间

3. **批量记录**：
   - 看病后立即记录就诊信息
   - 关联添加用药记录
   - 一次性完整记录

### 数据管理
1. **定期查看**：
   - 健康Tab快速查看所有健康数据
   - 体温趋势一目了然
   - 疫苗计划不遗漏

2. **下拉刷新**：
   - 所有列表都支持下拉刷新
   - 保持数据最新

3. **详细备注**：
   - 每条记录都可以添加备注
   - 记录特殊情况
   - 方便后续查询

### 里程碑记录建议
1. **及时记录**：
   - 宝宝有新变化立即记录
   - 第一时间的感受最珍贵

2. **拍照留念**：
   - 为重要里程碑添加照片
   - 记录成长瞬间
   - 打造专属成长相册

3. **详细描述**：
   - 记录当时的情景
   - 记录父母的心情
   - 为宝宝留下美好回忆

## ❓ 常见问题

**Q1: 体温记录能修改吗？**
A: 目前支持删除，编辑功能将在后续版本添加。

**Q2: 疫苗提醒如何设置？**
A: 添加疫苗时设置"下次接种日期"并开启提醒开关，系统会在接种前7天推送通知。

**Q3: 照片存在哪里？**
A: 当前存储在本地设备，建议定期备份。云端存储功能将在后续版本添加。

**Q4: 数据会同步到云端吗？**
A: 基础功能的同步已实现，新功能的云端同步正在开发中。

**Q5: 能导出这些数据吗？**
A: 数据导出功能正在开发中，将支持导出为Excel/PDF格式。
