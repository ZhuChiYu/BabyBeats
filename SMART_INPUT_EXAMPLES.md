# 智能输入功能示例

## 功能概述

智能输入功能允许用户使用自然语言描述宝宝的日常活动，系统会自动识别并生成对应的记录。无需逐个填写表单，大大提高记录效率。

## 支持的记录类型

- 🍼 **喂养记录**：母乳（左侧/右侧/双侧）、瓶喂母乳、配方奶
- 😴 **睡眠记录**：睡眠时段、时长
- 🚼 **尿布记录**：大便、小便
- 🍶 **挤奶记录**：单侧/双侧电动、手动

## 示例输入

### 示例 1：混合喂养记录

**输入：**
```
十点左右吃了60ml奶粉。12点吃了母乳20分钟。两点半吃奶粉80ml
```

**识别结果：**
- ✅ 10:00 - 配方奶 60ml (置信度: 高)
- ✅ 12:00 - 左侧母乳 20分钟 (置信度: 高)
- ✅ 14:30 - 配方奶 80ml (置信度: 高)

---

### 示例 2：喂养和尿布混合记录

**输入：**
```
昨晚八点多拉一次 吃到九点。12点吃完母乳又拉一次
```

**识别结果：**
- ✅ 昨天 20:00 - 便 (置信度: 高)
- ✅ 昨天 20:00-21:00 - 左侧母乳 (置信度: 中)
- ✅ 今天 12:00 - 左侧母乳 (置信度: 高)
- ✅ 今天 12:00 - 便 (置信度: 高)

---

### 示例 3：昨天凌晨的记录

**输入：**
```
今早四点左右又拉了一次 量不多
```

**识别结果：**
- ✅ 今天 04:00 - 便 (量不多) (置信度: 高)

---

### 示例 4：睡眠和喂养记录

**输入：**
```
七点半吃了睡到九点半。十点吃左边母乳18分钟。
```

**识别结果：**
- ✅ 07:30 - 左侧母乳 (置信度: 高)
- ✅ 07:30-09:30 - 睡眠 2小时 (置信度: 高)
- ✅ 10:00 - 左侧母乳 18分钟 (置信度: 高)

---

### 示例 5：详细的一天记录

**输入：**
```
早上6点吃了左边母乳15分钟。8点拉了一次大便。
9点吃了80ml配方奶，睡到11点。
下午1点吃右边母乳20分钟。3点又拉了，量很多。
傍晚6点吃了100ml奶粉。
```

**识别结果：**
- ✅ 06:00 - 左侧母乳 15分钟 (置信度: 高)
- ✅ 08:00 - 便 (置信度: 高)
- ✅ 09:00 - 配方奶 80ml (置信度: 高)
- ✅ 09:00-11:00 - 睡眠 2小时 (置信度: 高)
- ✅ 13:00 - 右侧母乳 20分钟 (置信度: 高)
- ✅ 15:00 - 便 (量很多) (置信度: 高)
- ✅ 18:00 - 配方奶 100ml (置信度: 高)

## 时间表达支持

### 相对时间
- `昨晚`、`昨天`、`今早`、`今天`、`凌晨`

### 中文时间
- `一点`、`两点`、`三点`...`十二点`
- `七点半` (30分钟)
- `十点左右`、`八点多` (模糊时间)

### 数字时间
- `10点`、`12点`
- `14:30`、`08:00` (24小时制)

---

## 数量和时长支持

### 奶量
- `60ml`、`80毫升`、`100ML`

### 时长
- `20分钟`、`1小时`、`2个小时`

### 睡眠时段
- `睡到九点半`
- `睡了2小时`
- `吃了睡到...`

## 使用技巧

### ✅ 推荐的描述方式

1. **明确时间点**
   ```
   十点吃了60ml奶粉  ✓
   ```

2. **自然语言描述**
   ```
   昨晚八点多拉一次，吃到九点  ✓
   ```

3. **指定左右侧**
   ```
   十点吃左边母乳18分钟  ✓
   ```

4. **添加备注信息**
   ```
   三点拉了，量很多  ✓
   ```

### ⚠️ 需要注意

1. **时间顺序**
   - 建议按时间顺序描述，系统会更准确识别

2. **置信度说明**
   - **高**：90%+ 准确率，可直接保存
   - **中**：70-90% 准确率，建议核对
   - **低**：<70% 准确率，请修改后重新解析

3. **编辑记录**
   - 如需修改，保存后可在对应页面编辑
   - 或修改原文本后重新解析

4. **一次性输入**
   - 可一次输入多天、多类型的记录
   - 系统会自动分类和排序

## 技术特点

### 🎯 智能识别
- 自动识别4种记录类型（喂养、睡眠、尿布、挤奶）
- 支持中文和数字混合时间表达
- 智能解析相对时间（昨天、今天）

### 📊 置信度评估
- 每条记录都有置信度标识
- 高置信度记录可批量保存
- 低置信度记录可单独删除

### 🔄 实时预览
- 输入后即时显示识别结果
- 可单独删除不需要的记录
- 一键批量保存所有记录

### 💾 数据安全
- 解析在本地完成，数据不上传
- 保存失败会提示具体原因
- 支持撤销操作
